# Turn-Based Rule Engine
### Engine-Agnostic Gameplay Systems Prototype (Python)

This project is a prototype of a turn-based RPG combat system designed to explore and validate gameplay mechanics before being migrated to a game engine.

The system is implemented in Python and focuses on modular architecture and rule-based combat logic. It includes a complete character editor and a tactical turn-based combat system inspired by tabletop and tactical RPG mechanics.

The project was developed as an engine-agnostic prototype to test gameplay systems independently from rendering and engine-specific features, with the long-term goal of transitioning the core logic to C# for integration with Unity.

---

## Features

- Modular turn-based combat system
- Character creation and attribute management
- Rule-based damage and combat resolution
- Engine-agnostic gameplay logic
- Designed for future migration to Unity (C#)

---

## Project Goals

- Prototype gameplay systems before engine integration
- Develop modular and maintainable system architecture
- Explore rule-based RPG combat mechanics
- Prepare the project for future migration to Unity

---

## Technologies

- Python
- Object-Oriented Programming (OOP)
- Modular system architecture
